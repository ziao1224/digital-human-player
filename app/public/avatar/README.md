# 2D 真人数字人素材准备指南

## 📁 文件结构

将真人形象图片放入此文件夹：

```
public/avatar/
├── base-face.png    # 基础脸部（必选）
├── left-eye.png     # 左眼（可选）
├── right-eye.png    # 右眼（可选）
├── mouth.png        # 嘴巴基础图（可选）
├── body.png         # 身体/衣服（可选）
└── README.md        # 本说明文件
```

## 🎨 图片规格要求

### 基础脸部 (base-face.png)
- **尺寸**: 400x400 像素
- **格式**: PNG（透明背景）
- **内容**: 正面照，包含完整的脸和脖子
- **眼睛**: 睁开状态
- **嘴巴**: 闭合或微笑状态
- **建议**: 使用专业证件照风格，光线均匀

### 左右眼 (left-eye.png, right-eye.png)
- **尺寸**: 100x80 像素
- **格式**: PNG（透明背景）
- **内容**: 单独的眼睛，睁开状态
- **用途**: 实现眨眼动画

### 嘴巴 (mouth.png)
- **尺寸**: 150x100 像素
- **格式**: PNG（透明背景）
- **内容**: 闭合状态的嘴唇
- **用途**: 通过缩放变形实现口型同步

### 身体 (body.png)
- **尺寸**: 300x400 像素
- **格式**: PNG（透明背景）
- **内容**: 上半身，职业装或正装

## 📸 拍摄建议

### 真人照片拍摄要求
1. **背景**: 纯色背景（白色或浅色）
2. **光线**: 正面均匀光，无阴影
3. **表情**: 自然微笑，眼神直视镜头
4. **角度**: 正面平视，肩膀水平
5. **服装**: 职业装，避免复杂图案
6. **分辨率**: 高清照片（至少 512x512）

### 后期处理
1. 使用 Photoshop 或在线工具（如 remove.bg）抠图
2. 确保边缘干净，无杂色
3. 调整亮度对比度，使面部清晰
4. 导出为 PNG 格式，保留透明通道
